  • 90

    Aba Estoque

    Acesse a aba do estoque para continuar o cadastro.

  • 91

    Guia Comunicador Satelital (Antena)

    Selecione a guia correspondente ao Comunicador Satelital.

  • 92

    Adicionar

    Clique no botão "Adicionar" para incluir um novo item.

  • 93

    Cadastrar Comunicador Satelital

    Inicie o cadastro do Comunicador Satelital preenchendo os dados básicos.

  • 94

    Descrição

    Olhe o contrato no ID Monitorado para obter as informações necessárias.

  • 95

    Número de Série

    Insira o código serial encontrado na parte de trás da antena.

  • 96

    Tipo de Equipamento

    Escolha "Comunicador Satelital" ou "Hidrosmart", dependendo do caso.

  • 97

    Fabricante

    Selecione "Globalstar", que é o fabricante da antena.

  • 98

    Modelo de Equipamento

    Informe o modelo como "SmartOne".

  • 99

    Status

    Certifique-se de que o status esteja definido como "Disponível".

  • 100

    Versão do Firmware

    A última versão é "Firmware 2.5.6".

  • 101

    Situação

    Defina a situação como "Primária".
