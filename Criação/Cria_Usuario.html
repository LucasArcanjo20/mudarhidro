Fluxo de Criação de Usuário

2ª Fase: Criação do Usuário

  • 1

    Aba Cliente

    Pesquise o nome do cliente na aba correspondente.

  • 2

    Pesquisar Cliente

    Procure pelo cliente no sistema usando o nome ou outro identificador.

  • 3

    Guia de Ações

    Acesse a guia de ações para gerenciar os registros do cliente.

  • 4

    Ícone Usuários

    Clique no ícone amarelo de usuários para continuar.

  • 5

    Adicionar Usuário

    Use a opção Adicionar para criar um novo usuário.

  • 6

    Cadastro Usuário

    Preencha as informações do usuário no formulário de cadastro.

  • 7

    Dados de Acesso

    Forneça login, e-mail, tipo de usuário e outros dados.

  • 8

    Dados Pessoais

    Inclua nome, RG, CPF, data de nascimento e telefone.

  • 9

    Endereço

    Adicione informações completas de endereço, incluindo CEP.

  • 10

    Situação

    Certifique-se de que a situação está marcada como "Habilitado".
