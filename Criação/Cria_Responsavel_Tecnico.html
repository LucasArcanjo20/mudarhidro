Criação de Responsável Técnico

Voltar para a Página Inicial

Este processo descreve como criar ou editar um Responsável Técnico, uma pessoa física habilitada para cobrir processos de produção e serviços em empresas.

Passos para Criar o Responsável Técnico

  • 1

    Aba Usuários

    Acesse a aba Usuários para iniciar o processo de criação do responsável técnico.

  • 2

    Aba Responsável Técnico

    Clique na guia Responsável Técnico para prosseguir com a criação ou edição.

  • 3

    Adicionar ou Editar

    Clique em Adicionar para criar um novo responsável técnico ou edite um existente.

  • 4

    Cadastro do Responsável Técnico

    Preencha os dados necessários do responsável técnico.

  • 5

    Número de Registro

    Importante: O número de registro é essencial para a validação do técnico junto ao conselho profissional.

  • 6

    Dados de Endereço

    Preencha o endereço completo com base nos documentos apresentados.

  • 7

    Dados de Acesso

    Preencha os dados de acesso, como login e senha. A senha deve ser entre 1 e 6 caracteres.

  • 8

    Revisão

    Revise todos os dados inseridos antes de finalizar o cadastro.

  • 9

    Teste de Acesso

    Teste o login e a senha para garantir que tudo está correto.

Detalhes Importantes

Certifique-se de que todos os campos estão preenchidos corretamente antes de finalizar o processo. Caso o responsável técnico já esteja cadastrado, utilize a funcionalidade de edição para fazer ajustes.

Atenção especial ao número de registro e órgão emissor, pois são essenciais para a validação do técnico.
