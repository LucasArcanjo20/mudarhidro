Fluxo de Criação de Monitorado

4ª Fase: Criação de Monitorado

  • 46

    Aba Monitorados

    Acesse a aba de Monitorados para iniciar o processo.

  • 47

    Adicionar

    Clique no botão "Adicionar" para incluir um novo Monitorado.

  • 48

    Cadastrar Monitorado

    Inicie o cadastro do Monitorado preenchendo os dados necessários.

  • 49

    ID Monitorado

    Informe o ID do Monitorado. Se houver dúvidas, consulte o caderno para mais informações.

    • Exemplo: HS01 – Hidrocontrol
    • HS: Hidrosat
    • 01: Número de equipamento dentro do cliente
    • 03: Nomenclatura do poço
  • 50

    Descrição

    Informe a descrição do Monitorado, geralmente fornecida.

  • 51

    Cliente

    Informe quem será o usuário do Monitorado.

  • 52

    Anotações

    Se houver alguma anotação relevante, inclua neste campo.

  • 53

    Endereço

    Informe o endereço completo para que a latitude e longitude possam ser preenchidas automaticamente.

  • 54

    Latitude

    Informe a latitude do local do Monitorado.

  • 55

    Longitude

    Informe a longitude do local do Monitorado.

  • 56

    Situação

    Defina a situação do Monitorado, geralmente como "Habilitado".
