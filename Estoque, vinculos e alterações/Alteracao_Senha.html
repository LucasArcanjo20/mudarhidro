Este guia descreve como realizar a alteração de senha no sistema, tanto para clientes quanto para responsáveis técnicos. Este processo é fundamental para garantir a segurança do sistema.

Passos para Alteração de Senha

    Acesse a aba Cliente

Observações Importantes

  • •

    Responsável Técnico

    Utiliza o login de acesso em vez da matrícula.

  • •

    Usuário

    A pessoa que utiliza o sistema com permissões específicas.

Após a alteração, teste o login com a nova senha para garantir que o processo foi concluído com sucesso.
