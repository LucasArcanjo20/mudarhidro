Gestão de Pacotes

Este guia descreve os passos para realizar a gestão de pacotes no sistema, incluindo a análise de dados de falso pulso associados aos monitorados.

Passos para a Gestão de Pacotes

  • 1

    Acesse a aba Gestão de Pacotes

    Clique na aba correspondente no sistema para começar.

  • 2

    Selecione o Cliente

    Escolha um cliente específico ou visualize todos os clientes juntos.

  • 3

    Selecione a Data

    Utilize o filtro de data para refinar os dados exibidos.

  • 4

    Execute a Operação

    Clique no botão "Executar" para carregar as informações desejadas.

  • 5

    Analise os Dados de Falso Pulso

    Revise os dados de falso pulso configurados na aba Monitorado.

Observações Importantes

  • !

    OBS: Certifique-se de verificar os dados na aba Monitorado antes de executar a análise.

  • !

    OBS: Sempre revise os dados exibidos antes de concluir o processo.
