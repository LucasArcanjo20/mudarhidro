Troca de Equipamento no Sistema

Voltar para a Página Inicial

Este processo orienta a troca completa de um equipamento por outro conjunto de equipamento no sistema. É importante que a remoção do equipamento anterior tenha sido finalizada antes de iniciar este procedimento.

Passos para Trocar o Equipamento

  • 1

    Certifique-se de que a remoção do equipamento antigo foi concluída.

    A remoção do equipamento anterior precisa estar finalizada para garantir que o novo equipamento será corretamente registrado no sistema.

  • 2

    Acesse a aba Estoque.

    No sistema, navegue até a aba de "Estoque" para continuar o processo de cadastro do novo equipamento.

  • 3

    Clique em Adicionar.

    Clique no botão "Adicionar" para iniciar o cadastro de um novo equipamento no sistema.

  • 4

    Preencha os dados do Equipamento

    Insira as informações do novo equipamento, como descrição, número de série, fabricante, modelo e status.

  • 5

    Clique em Cadastrar.

    Após preencher todos os campos, clique em "Cadastrar" para salvar o novo equipamento no sistema.

  • 6

    Ajuste o status do Medidor.

    Na guia "Medidor", altere o status de "Revisão" para "Disponível" para finalizar o processo de troca do medidor.

  • 7

    Criação do comunicador satelital.

    Crie o comunicador satelital na guia correspondente, preenchendo todos os dados necessários, como número de série e fabricante.

Observações Importantes

  • OBS: Sempre confirme o ID Monitorado e os números de série antes de concluir a troca do equipamento.
  • Nota: O cadastro do equipamento no sistema é um procedimento fundamental para garantir a rastreabilidade.
