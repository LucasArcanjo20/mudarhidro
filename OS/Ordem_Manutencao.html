Ordem de Manutenção

Este guia detalha o processo para realizar uma ordem de manutenção no sistema, que inclui modificar e revisar equipamentos. Siga as etapas abaixo para garantir que o procedimento seja realizado corretamente.

Passos para Criar a Ordem de Manutenção

  • 1

    Acesse a aba Ordem de Serviços

    Navegue até a aba correspondente para iniciar o processo.

  • 2

    Clique em Cadastrar

    Inicie a ordem de manutenção clicando no botão apropriado.

  • 3

    Selecione o Tipo de Operação

    Defina o tipo como Manutenção.

  • 4

    Revise os Dados

    Verifique se as informações estão corretas antes de prosseguir.

Observações Importantes

  • OBS 1: Troque o número de série de equipamentos apenas durante a etapa final da OS.
  • OBS 2: Certifique-se de que todos os campos estão preenchidos corretamente antes de finalizar.
